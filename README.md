📌 Credit Risk Analysis and Loan Default Prediction

🎯 Project Overview
This project analyzes customer financial data to identify high-risk customers and understand factors affecting loan default using Excel, SQL, and Power BI.

🛠️ Tools Used
Excel – Data Cleaning
SQL – Data Analysis
Power BI – Dashboard & Visualization

📊 Key Analysis
Loan Status Distribution
Credit Score Analysis
Income Analysis
Loan Purpose Analysis
Home Ownership Distribution

🔍 Key Insights
Most customers are non-default
Higher credit scores reduce default risk
Higher income customers are less likely to default
Debt consolidation is the most common loan purpose
